# LTracker

Tracker pessoal minimalista para treino, sono e recuperacao.

## Objetivo

Registrar rapidamente dados do dia, atividades e um resumo semanal simples, usando apenas recursos locais do navegador.

## Stack

- HTML
- CSS
- JavaScript puro
- localStorage

## Como usar

Abra o arquivo `index.html` no navegador.

## Estrutura

```text
LTracker/
  index.html
  style.css
  app.js
  docs/
    DEV_LOG.md
    DATA_SCHEMA.md
    ROADMAP.md
```

## Principios

- MVP primeiro
- Simplicidade
- Baixa friccao
- Sem backend
- Sem frameworks
- Sem build tools
